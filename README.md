# seeeduino_xiao_serial

## 概要

SeeeduinoXiaoでシリアル通信するRustコードです。

### ソフトウェア

* teratermでSeeduinoXiaoと接続されたシリアルポートを開き、以下の設定をします
    * シリアルポートオープン
    * 設定>シリアルポート設定>スピード:115200
    * 設定>端末>改行コード:AUTO
